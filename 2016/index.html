Codeurs En Seine 2016
se déroulera le
{{ site.date[page.group] }} // Université de Rouen


Codeurs en Seine est une conférence gratuite qui se déroule à Rouen, sur 4 thèmes en simultané : Web, Agile, Java et Technologies.

La conférence est organisée bénévolement par Normandie Web Xperts, Normandy Java User Group, Normandy Agile User Group et l'Université de Rouen ; elle est également une étape de l'agile tour.
