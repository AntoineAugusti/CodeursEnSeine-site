Codeurs en Seine est une conférence gratuite qui se déroule à Rouen, sur 4 thèmes en simultané : le Java, l'Agile, le Web et l'Innovation. Vous pourrez défricher des sujets comme les bases de données NoSQL, l'histoire du web, Java 8, les objets connectés et tout se qui tourne autour du cloud.

Rendez-vous au Madrillet dans les locaux de l'Université de Rouen, UFR des Sciences et Techniques, avenue de l'université à Saint-Étienne-du-Rouvray. Les conférences se tiendront toute la journée 9h00 à 18h00. 5 sessions longues et 2 sessions courtes, entrecoupées de pauses gastronomiques, se tiendront par salle.

La conférence est organisée bénévolement par Normandy Web Xperts, le Normandy Agile User Group, Le Normandy Java User Group et l'Université de Rouen ; elle est également une étape de l'agile tour.

La conférence sera suivie d'une soirée communautaire.
