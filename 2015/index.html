Codeurs En Seine 2015

{{ site.date[page.group] }} // Université de Rouen

Codeurs en Seine est une conférence gratuite qui se déroule à Rouen, sur 4 thèmes en simultané : le Java, l'Agile, le Web et l'Innovation.

Rendez-vous le {{ site.date[page.group] }} au Madrillet dans les locaux de l'Université de Rouen, UFR des Sciences et Techniques, avenue de l'université à Saint-Étienne-du-Rouvray.

La conférence est organisée bénévolement par Normandy Web Xperts, le Normandy Agile User Group, Le Normandy Java User Group et l'Université de Rouen ; elle est également une étape de l'agile tour.
