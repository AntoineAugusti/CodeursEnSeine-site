Organisateurs

Normandy Java User Group, Normandy Agile User Group, Normandie Web Xperts et l'université de Rouen mettent à profit leur expérience et leurs contacts pour proposer un événement unique sur les bords de Seine.

Leur volonté est de réunir sur une journée les codeurs d'ici et d'ailleurs afin qu'ils partagent leurs connaissances.
